+ 此系统实现了姓名与成绩自动生成的功能

+ 实现了每个人所有科目的分数总和的功能
+ 实现了每科科目的平均值的功能
+ 实现了按照总分进行排名的功能
+ 按照排名后的位置将学生信息写入到一个文件中
